ScribeUI

ScribeUI is a cloud based tool to easily edit mapfiles and immediatly view the result, all at the same place.

  • Code and preview your map at the same place
  • Cloud-based
  • Easily work on specific areas of the map with points of interest
Install ScribeUI and get started!

Scribe

Scribe is an alternative syntax to make mapfile development faster and easier.

  • Easy scale definition
  • Variables in your mapfile
  • Multi-line comments
  • Generates a valid and nicely indented mapfile
Learn more about the Scribe syntax!
